# SaturdAIfever Homebrew Tap

This public tap is the official Homebrew distribution channel for
SaturdAIfever command-line software.

The Portico cask will install the proprietary compiled binary published by
[`saturdaifever/portico-releases`](https://github.com/saturdaifever/portico-releases).
It will not build or distribute Portico source code.

## Portico

The installation command will become active with the first public Portico
binary release:

```sh
brew install --cask saturdaifever/tap/portico
```

After installation, upgrades will use the standard Homebrew flow:

```sh
brew upgrade --cask portico
```
